# Quick QR-Code Generator

Minimal QR code generator. Paste a link, get a QR code instantly.

## Usage

Paste a URL → QR appears immediately. Type manually → press Enter or the arrow button.

Click the QR code to copy the original link. Click Download to save.

**Advanced mode** (bottom-right) unlocks colors, dot styles, error correction, size, quiet zone, logo upload, Wi-Fi / email / phone content types, and export format.

## Dev

```bash
npm install
npm run dev
npm run build
```

## Stack

- Vite + React
- [qr-code-styling](https://github.com/kozakdenys/qr-code-styling)
